# E-Commerce-Sales-Dashboard
Interactive E-Commerce Sales Performance Dashboard using Power BI | 4th Semester Project - Chandigarh University (April 2026)

Second Year Project
Chandigarh University
April 2026

Project Overview
An interactive Power BI dashboard that analyzes 3,500+ e-commerce sales records (2022–2024) to provide real-time insights on sales, profit, regional performance, and product-wise trends. The dashboard helps business stakeholders make data-driven decisions.

Dataset Source: Kaggle - E-Commerce Sales Analysis (Junior Analyst)

Key Metrics:

Total Sales: ₹1,06,67,881
Total Profit: ₹18,44,665
Total Orders: 3,500+
Regions: North, South, East, West
Categories: Electronics, Accessories, Office Supplies
Dashboard Features
Interactive slicers (Year, Category, Region)
Sales & Profit by Region (Donut charts)
Sales by Category & Product
Monthly trend analysis
Top performing products
Profit margin analysis
Folder Structure
/Data → Raw dataset
/PowerBI → Main .pbix file

How to Use
Download the .pbix file from /PowerBI
Open it in Power BI Desktop (free)
Explore all pages and interact with slicers
Future Enhancements
Predictive sales forecasting
